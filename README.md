# 🚀 MindArchive Chrome Extension – Setup Guide

Follow these steps to install and use the extension locally.

---

## 📥 1. Download the Extension

1. Go to the GitHub repository
2. Click on **Code → Download ZIP**
3. Extract the ZIP file on your system

---

## 📂 2. Locate Extension Folder

After extracting, open:

```
MindArchive-ChromeExtension
```

👉 This is the folder you will load into Chrome

---

## 🌐 3. Open Chrome Extensions Page

1. Open Chrome
2. Go to:

```
chrome://extensions/
```

---

## ⚙️ 4. Enable Developer Mode

* Toggle **Developer Mode** (top right corner)

---

## ➕ 5. Load the Extension

1. Click **Load unpacked**
2. Select the folder:

```
MindArchive-ChromeExtension
```

---

## ✅ 6. Start Using

1. Open any website
2. Click the **MindArchive extension icon**
3. Click **Save**

---

## 🔐 7. Login Requirement

* If you see **"Login required"**
  👉 Click **Go to Login →**
  👉 Login on the website

---

## ⏳ 8. Important Note (Free Hosting Delay)

* Backend is hosted on free tier (Render)
* It may take **~30–60 seconds** to wake up

👉 If content doesn’t save instantly:

* Wait a few seconds
* Try again

---
    
## 🧠 How it works (simple)

```
Extension → API → Queue (Redis) → Worker → Database
```

---

## 🎯 Done!

Your extension is now ready to use 🎉
